Carmudgeon

Ruby for QT with a rails like MVC syntax and support for a local sql database

Download .zip Download .tar.gz View on GitHub

Welcome to Carmudgeon a Ruby QT application boilerplate.

This is an app skeleton for ruby projects that use Nokia's QT GUI toolkit. It has some prebuilt naming conventions and file structure that supports the MVC scheme.
